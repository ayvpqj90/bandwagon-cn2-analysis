## CN2 是什么，GT 和 GIA 有什么本质区别

简单说：**CN2（CTGNet）是中国电信的专属骨干网，分两个等级——GT（Global Transit）和 GIA（Global Internet Access）。**

GT 是中低端，去程走的是普通 163 骨干网，只有回程才走 CN2 骨干，晚高峰会有一定程度的拥堵。GIA 则是双程全走 CN2 骨干，不管是从国内连出去还是数据回来，全程走高端线路，晚高峰也基本不受大盘流量影响。

GIA-E（ECOMMERCE）是搬瓦工的术语，指的是 GIA 线路的电商优化版本，具体就是 DC6 和 DC9 两个洛杉矶机房接入的 CN2 GIA，带宽从 2.5Gbps 起步，最高可以跑到 10Gbps。
